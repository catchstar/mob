用法

HTML 标签

你可以仅使用HTML代码来实现tab切换效果，不需要写一行js代码。你只需要在html标签里 标记data-toggle="tab"就能显示 mob样式的tab。

可调用的方法

事件

.show
.shown
$('a[data-toggle="tab"]').on('shown', function (e) { e.target // activated tab e.relatedTarget // previous tab })

最佳实践

你也可以配合url routers来使用tab.

Template

mob 框架不内置 template 功能，推荐你使用其它框架的相关功能，如 backbones的template功能。

Router

mob 框架不内置 Router 功能，推荐你使用其它框架的相关功能，如 backbones的 Router功能。
